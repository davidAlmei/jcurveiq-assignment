# JcurveIQ Frontend Assignment

## Overview
Recreated the Research Workspace UI using React (Vite), Tailwind CSS and Redux Toolkit.

## Live Demo
[https://your-vercel-url.vercel.app](https://jcurveiq-assignment-rho.vercel.app/)

## Setup
1. `npm install`
2. `npm run dev`

## Toggle mock states
Open `src/features/projectsSlice.js` and set `simulateError` to `true` to simulate an error.

## Folder structure
src/

  features/
  components/
  pages/
public/mock/projects.json

## Tradeoffs
Used local mock data; focused time on accessibility, responsiveness and polish.
